Desarrollo Frontend

El Frontend es la parte de una aplicacion web que los usuarios pueden ver e interactuar directamente.

Tecnologías principales

  1. HTML

    2. Define la estructura del contenido de una página web.

  2. CSS

    3. Permite controlar la apariencia visual del sitio.

  3. JavaScript

    4. Permite crear interfaces interactivas.
